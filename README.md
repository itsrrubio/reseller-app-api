\# Reseller App API



A backend system built with .NET 8 and ASP.NET Core Web API to manage inventory, listings, and sales across multiple reselling platforms like eBay, Mercari, and Poshmark.



\## Tech Stack

\- .NET 8

\- ASP.NET Core Web API

\- Entity Framework Core

\- SQL Server

\- Swagger



\## Features (Current)

\- Inventory management

\- Item creation with multiple images

\- RESTful API endpoints

\- EF Core relational database design



\## Coming Soon

\- JWT Authentication

\- Role-based authorization

\- Order management

\- Pricing \& profit tracking

\- Analytics dashboard



\## Purpose

Built to streamline and automate a real-world reselling business workflow and serve as a scalable foundation for future integrations and AI-driven automation.
